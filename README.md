> Features

- wildcard entries
- nested directories
- css without needed import in javascript
- json hashmap files for js, css and assets
- compressing assets
- source maps, editable sourceMappingURL
- sass, postcss, autoprefixer
- es6 transpiling
- clean rebuild
- vue

<br>

### Installation

```sh
npm install
```

### Development

```sh
npm run watch
```

### Production

```sh
npm run build
```
